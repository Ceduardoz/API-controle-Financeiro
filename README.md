<h2 id="about">📌 Sobre</h2>

API REST para um **Sistema de Controle Financeiro**, desenvolvida com **Node.js, Express, Prisma e PostgreSQL**.

A API permite:

- Criar e gerenciar **transações financeiras**
- Registrar **receitas e despesas**
- Organizar dados por **categorias**
- Consultar histórico de movimentações
- Integrar com aplicações **Front-end (React)**

Este projeto faz parte de um estudo prático de **arquitetura backend e APIs REST**.
